# Daily Practice Routine

Follow this structure every day to build consistent progress.

## Daily Commitment: 30-60 minutes

### Morning Routine (15-20 min)
1. **Review yesterday's notes** (5 min)
   - Read previous day's log
   - Review any commands you struggled with

2. **LabEx Daily Challenge** (10-15 min)
   - Complete one interactive lesson
   - URL: https://labex.io/learn/linux

### Main Practice Session (30-40 min)

#### Option A: Structured Learning (Mon/Wed/Fri)
1. **Study current week's topic** (15 min)
   - Read topic guide in `/topics`
   - Watch relevant video section if needed

2. **Hands-on Practice** (15 min)
   - Complete exercises from current week
   - Use LabEx or local terminal

3. **Document** (10 min)
   - Update daily log
   - Add to personal cheat sheet

#### Option B: Challenge Day (Tue/Thu)
1. **Solve a challenge** (20 min)
   - From LabEx practice challenges
   - Or create your own scenario

2. **Research & Learn** (10 min)
   - Look up commands you didn't know
   - Read man pages

3. **Document** (10 min)
   - Write what you learned
   - Update achievements

#### Option C: Project Day (Sat/Sun)
1. **Work on a mini-project** (30 min)
   - Build something practical
   - Automate a task
   - Set up a service

2. **Document** (10 min)
   - Write about your project
   - Update GitHub

### Evening Review (5 min)
1. **Update achievements** - Check off completed items
2. **Plan tomorrow** - Note what you'll focus on
3. **Commit to GitHub** - Push your daily log

## Weekly Milestones

### End of Week Checklist
- [ ] Completed 5+ practice sessions
- [ ] Finished week's exercises
- [ ] Updated achievements
- [ ] Created/updated cheat sheet
- [ ] Pushed all changes to GitHub

### Weekly Review (30 min on Sunday)
1. Review all daily logs from the week
2. Test yourself on the week's commands
3. Complete week's achievement checklist
4. Plan next week's focus areas

## Monthly Goals

- Complete 4 weekly milestones
- Unlock next level achievements
- Build 1 practical project
- Write 1 tutorial or guide

- **Small wins matter**: Even 15 minutes counts
