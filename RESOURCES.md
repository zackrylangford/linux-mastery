## Documentation & References

### Linux Handbook
- **URL**: https://linuxhandbook.com/
- **What**: Practical guides and tutorials
- **Use for**: Quick references, how-tos

### Man Pages (Built-in)
- **Command**: `man <command>` or `man -k <keyword>`
- **What**: Official documentation for every command
- **Use for**: Deep dives, understanding options

### TLDR Pages
- **Install**: `sudo apt install tldr` or `npm install -g tldr`
- **What**: Simplified, practical examples
- **Use for**: Quick command usage
