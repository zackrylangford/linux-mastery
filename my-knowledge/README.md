This is where you document:

3. **Daily learning notes** - Quick logs of what you learned

## How to Use This

**After each learning session:**
1. Add the problem to `problems-i-solve.md`
2. Save any good AI prompts to `good-prompts.md`
3. Quick note in `daily-log.md` (optional)

**When facing a new problem:**
1. Check `problems-i-solve.md` - have you solved something similar?
2. Check `good-prompts.md` - do you have a prompt template?
3. If not, learn it and add it!
