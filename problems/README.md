## How to Use This Directory

1. **Pick a problem** you want to solve (e.g., "I need to find large files")
2. **Open that category** and read the problem guide
3. **Learn what's possible** - what tools exist and when to use them
4. **Practice recognition** - look at examples and understand what they do
5. **Try with AI** - use the prompt templates to generate solutions
6. **Document** - add what you learned to `my-knowledge/`

## Problem Guide Structure

Each problem guide follows this format:
- **What problem does this solve?** - Clear description
- **When do you encounter this?** - Real-world scenarios
- **Available approaches** - Tools and their tradeoffs
- **Examples to recognize** - Code snippets with explanations
- **Try with AI** - Prompt templates to practice
