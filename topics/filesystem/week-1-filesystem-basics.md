# Week 1: Filesystem Basics

## Core Concepts
- Linux filesystem hierarchy (/, /home, /etc, /var, /usr, /bin, /tmp)
- Absolute vs relative paths
- Hidden files and directories
- Inodes and links

## Commands to Master
- `pwd` - Print working directory
- `ls` - List directory contents (with -l, -a, -h flags)
- `cd` - Change directory
- `mkdir` - Create directories
- `rmdir` - Remove empty directories
- `touch` - Create files
- `cp` - Copy files/directories
- `mv` - Move/rename files
- `rm` - Remove files (careful with -rf)
- `find` - Search for files
- `tree` - Display directory structure

## Daily Exercises
1. Navigate the entire filesystem hierarchy
2. Create a complex directory structure
3. Practice with symbolic and hard links
4. Use find to locate specific files
5. Understand file metadata with stat

## Resources
- `man hier` - Filesystem hierarchy description
- `man 7 file-hierarchy` - Modern filesystem layout
